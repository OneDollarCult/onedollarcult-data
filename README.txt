OneDollarCult — Технический пакет

1. players.json — список всех участников
2. fetchPlayers.js — скрипт для вставки в Tilda, чтобы показывать счётчик игроков
3. rules.md — текст для страницы "Правила"
4. philosophy.md — текст для "Философии"
5. socials.md — текст для "Соцсети"
6. Как опубликовать:
   - Создать GitHub репозиторий
   - Загрузить всё содержимое архива
   - Включить GitHub Pages: Settings → Pages → выбрать "main" и папку root
   - Получишь ссылку вида https://username.github.io/players.json
